# Eunilite — Product Roadmap

## Phase 1 — Foundation (Weeks 1–8)
Get the core experience working end to end.

### Deliverables
- [ ] React Native mobile app (iOS + Android)
- [ ] Next.js web app
- [ ] Node.js backend API
- [ ] PostgreSQL database schema
- [ ] Group creation and management
- [ ] Member invites and profiles
- [ ] Contribution tracking (manual confirmation)
- [ ] Payout schedule and rotation tracker
- [ ] Basic notifications (push + email)
- [ ] Treasurer dashboard

---

## Phase 2 — Payments (Weeks 9–14)
Make money move inside the platform.

### Deliverables
- [ ] Ozow instant EFT integration
- [ ] Peach Payments debit card
- [ ] SnapScan QR payments
- [ ] Automated debit orders
- [ ] Instant bank payouts
- [ ] Payment receipts and references
- [ ] Dispute management flow

---

## Phase 3 — Intelligence (Weeks 15–20)
Turn data into financial power for members.

### Deliverables
- [ ] Savings credit score algorithm
- [ ] Advanced reporting and exports
- [ ] WhatsApp Business API bot
- [ ] Late penalty automation
- [ ] Interest-bearing holding account (money market)
- [ ] Group buying discount partnerships

---

## Phase 4 — Scale (Month 6+)
Grow beyond individual stokvels.

### Deliverables
- [ ] Burial society module
- [ ] Multi-group management for organisations
- [ ] White-label offering for banks and retailers
- [ ] API for third-party integrations
- [ ] Credit scoring partnerships with lenders
- [ ] iOS App Store + Google Play launch
